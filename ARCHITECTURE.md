## 分析路由矩阵

| 标的类型 | 识别规则 | 分析路径 | 可用模式 |
|---------|----------|---------|---------|
| **A股** | 6位数字，首字0/3/6 | 6维全量 | 默认全量（可选 --valuation / --timing / --risk 专项） |
| **ETF** | 6位数字，首字1/5 | 3维(技术+情绪+溢价) | 默认全量（可选 --timing，无valuation/risk因无财报） |
| **港股** | 5位数字 | 6维(数据有限，情绪面/风控部分缺失) | 默认全量（可选专项，部分A股特色数据源不可用） |
| **美股** | 纯字母 | 6维(需FMP Key) | 默认全量（可选 --valuation / --timing / --risk 专项） |

## 六维分析详解

### 维度1: 基本面 — 杜邦分析 (dupont.py)

```
输入: 多期财报数据 (5年)
      │
      ▼
┌─────────────────────────────────┐
│ 三因素分解                        │
│ ROE = 净利率 × 周转率 × 杠杆      │
│                                  │
│ 五因素分解                        │
│ ROE = 税负 × 利息负担 × 营业利润率 │
│     × 周转率 × 杠杆               │
│                                  │
│ 驱动因素识别                      │
│ → 利润率驱动(品牌型)              │
│ → 周转率驱动(零售型)              │
│ → 杠杆驱动(金融型)                │
│                                  │
│ ROE质量评级 (0-8分)               │
│ · ROE水平 (0-3分)                │
│ · 净利率 (0-2分)                  │
│ · 杠杆安全 (0-2分)                │
│ · 趋势方向 (0-1分)                │
└─────────────────────────────────┘
输出: {latest, periods, roe_trend, driver, quality}
```

### 维度2: 估值 (valuation.py)

```
输入: 当期财报 + 市场数据 + 多期数据
      │
      ├──▶ DCF两阶段
      │    FCFF = EBIT×(1-税率) + 折旧 - 资本支出
      │    5年预测期 + 永续增长
      │    WACC = 10%, Terminal Growth = 3%
      │    → 内在价值/股
      │
      ├──▶ 蒙特卡洛 (10,000次)
      │    增长率 ~ N(历史增长率, 5%)
      │    WACC ~ N(10%, 1.5%)
      │    永续 ~ U(2%, 4%)
      │    → P5/P10/P25/P50/P75/P90/P95 分布
      │
      ├──▶ 反向DCF (二分法)
      │    给定当前市值 → 反推隐含增长率
      │    → "市场预期年化X%增长"
      │
      ├──▶ 格雷厄姆数值
      │    √(22.5 × EPS × BVPS)
      │    → 价值投资底线
      │
      ├──▶ 相对估值
      │    PE/PB vs 行业均值 → 溢价/折价百分比
      │    （需 industry_peers 数据，非所有市场可用）
      │
      └──▶ 安全边际
           (内在价值 - 当前价) / 内在价值
           → deep_value / undervalued / slightly_undervalued
             / slightly_overvalued / overvalued
```

### 维度4: 风控 (risk_screening.py)

```
输入: 2期+财报数据 + 市场数据 + 质押数据
      │
      ├──▶ Beneish M-Score (财报操纵检测)
      │    8因子概率模型:
      │    DSRI(应收占比) + GMI(毛利率) + AQI(资产质量)
      │    + SGI(收入增长) + DEPI(折旧) + SGAI(管理费)
      │    + TATA(应计利润) + LVGI(杠杆)
      │    M = -4.84 + 0.92×DSRI + ... + 4.679×TATA - 0.327×LVGI
      │    阈值: M > -1.78 → 疑似操纵
      │
      ├──▶ Altman Z-Score (破产预警)
      │    5因子: 流动性 + 累积盈利 + 资产回报 + 杠杆容忍 + 周转效率
      │    Z > 2.99 安全区 | 1.81-2.99 灰色区 | < 1.81 困境区
      │
      ├──▶ Piotroski F-Score (财务健康)
      │    9项是/否检查: ROA正/OCF正/ROA上升/盈利质量/
      │    杠杆下降/流动性改善/未稀释/毛利率上升/周转率上升
      │    ≥ 8强买入 | 6-7偏多 | 4-5中性 | 2-3偏空 | ≤ 1强卖出
      │
      └──▶ A股特色8项检查
           ┌──────────────────────────────────────────┐
           │ 检查项              阈值        红灯/黄灯  │
           │ ──────────────────────────────────────── │
           │ 应收/营收比         >30%/>20%    红/黄     │
           │ 现金流/净利润       <0.7连续2年   红/黄     │
           │ 商誉/净资产         >30%/>15%    红/黄     │
           │ 资产负债率          >70%/>60%    红/黄     │
           │ 流动比率            <1.0/<1.5    红/黄     │
           │ 存贷双高            现金>15%且负债>15%  黄 │
           │ 自由现金流          <0          黄        │
           │ 质押比例            >80%/>50%    红/黄     │
           │ ──────────────────────────────── │
           │ 评分: 红灯×20 + 黄灯×8 = 0-100    │
           └──────────────────────────────────────────┘
```

### 维度6: 综合评分 (comprehensive_score.py)

```
输入: 5个维度分析结果
      │
      ├──▶ 各维度独立评分 (0-100)
      │    基本面: ROE质量分 × 10 + 20保底
      │    估值:   安全边际 → 阶梯映射
      │    技术:   择时信号 × 0.7 + 置信度 × 0.3
      │    情绪:   直接取综合情绪分
      │    风控:   100 - 风险分 (反向)
      │    温度:   直接取综合情绪分
      │
      └──▶ 加权合成 → 操作建议
           ┌───────────────────────────────────┐
           │  A股评分权重:                      │
           │  基本面 25% + 估值 20% + 情绪 20%  │
           │  + 技术 15% + 风控 10% + 温度 10% │
           │                                    │
           │  ETF评分权重:                      │
           │  技术 35% + 情绪 30% + 溢价 35%   │
           │                                    │
           │  A股: ≥80优秀 | ≥60良好 | ≥40一般 | <40较差 │
           │  ETF: ≥75优秀 | ≥55良好 | ≥35一般 | <35较差 │
           │                                    │
           │  操作建议决策树 (按优先级):             │
           │  安全边际>30% + 买信号 → 建仓(高)     │
           │  安全边际>30% + hold  → 等待买点(中高) │
           │  安全边际>15% + 买信号 → 加仓(中高)   │
           │  安全边际<0%  + 卖信号 → 减仓(高)     │
           │  安全边际<-15%        → 规避(高)      │
           │  总分≥60              → 持有(中)      │
           │  总分≥40              → 观望(中)      │
           │  其他                 → 减持(中)      │
           └───────────────────────────────────┘
```

## 关键设计决策

| 决策 | 理由 |
|------|------|
| 纯Python手写DCF/蒙特卡洛 | 不引入QuantLib/OpenBB等重依赖 |
| 技术指标双实现(pandas-ta+手写) | pandas-ta缺失时自动降级 |
| ETF溢价三通道降级 | 雪球token过期→东财→新浪，保证可用性 |
| 缓存默认禁用 | 保证每次分析使用最新数据，可按需启用 |
